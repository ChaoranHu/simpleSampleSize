# README #


* This package provide a simple and straight implementation for sample size calcualtion.

* Version: 1.0.0.9000


## The scope of this package

For each type of variable(endpoint), this package provides functions to calculate power
(if sample size specified) or sample size (if power specified). The types of variables
includes

- continuous
  - t-test
  - z-test
- binary
  - normal approximation
  - Fisher test
- count
  - count data with over-dispersion
